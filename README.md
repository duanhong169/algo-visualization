# React + TypeScript + Tailwind Template

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check + production build |
| `npm run lint` | Run ESLint |
| `npm run lint:fix` | Auto-fix lint issues |
| `npm run format` | Format with Prettier |
| `npm run type-check` | TypeScript type checking |
| `npm run test` | Run tests |
| `npm run test:watch` | Run tests in watch mode |

## Claude Code Commands

| Command | Description |
|---------|-------------|
| `/component` | Generate a new React component with standard structure |
| `/hook` | Generate a custom hook with tests |
| `/review` | Run code review checklist |
| `/refactor` | Analyze and refactor code |
